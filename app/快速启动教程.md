# 行动安全守护系统 — 移动端快速启动教程

面向零基础用户。10 分钟内在手机上看到移动端 App。

---

## 第一步：确保后端服务已启动

移动端 App 需要后端提供数据。请先按照 Web 端快速启动教程启动后端服务。

```bash
# 在 web/ 目录下
npm run build
npm run server
```

看到「服务已启动: http://localhost:3001」说明后端 OK。

---

## 第二步：安装移动端依赖

在 `app/` 目录下打开 cmd，输入：

```bash
npm install
```

---

## 第三步：启动移动端开发服务器

```bash
npm run dev
```

看到类似输出：

```
VITE v8.x.x  ready in xxx ms
➜  Local:   http://localhost:5174/
➜  Network: http://192.168.1.xxx:5174/
```

---

## 第四步：手机访问

1. 手机和电脑连同一 Wi-Fi
2. 手机浏览器输入 `http://192.168.1.xxx:5174`（用你电脑的实际 IP）
3. 看到登录页即可

> 电脑 IP 查看方式：Win + R → `cmd` → `ipconfig` → 找到「IPv4 地址」

---

## 第五步：安装到手机桌面（PWA）

1. 用手机浏览器打开地址
2. **Chrome**：右上角三个点 → 添加到主屏幕
3. **Safari**：底部分享按钮 → 添加到主屏幕
4. 手机会出现一个图标，下次点击直接打开（无需输网址）

## 常见问题

### Q: 手机访问显示"无法连接"

1. 确认电脑和手机连同一 Wi-Fi
2. 确认 `npm run dev` 窗口还开着
3. 确认 IP 地址是最新的（`ipconfig` 重新查看）
4. 检查 Windows 防火墙是否拦截 Node.js

### Q: 登录后页面空白

确认后端服务（`web/server/`）已启动在 3001 端口，移动端会自动代理 API 请求。

### Q: 验证码在哪看？

Mock 模式下，验证码打印在运行 `npm run server` 的黑色窗口中。`[Email Mock]` 或 `[SMS Mock]` 开头的行里 6 位数字。

### Q: 底部导航栏不见了？

只有登录后（数据看板/告警/设备/守护页面）才会显示底部 Tab。登录页和注册页是独立页面，没有底部导航栏。

### Q: 怎么打包成真正的 APK？

需要 Android Studio 环境，详见项目 README 中的「Capacitor 打包」章节。
